# Hextech E-commerce

E-commerce web application developed as a capstone project (2022), applying concepts of programming logic, data organization, and full-stack web development.

## Features

- Product catalog
- User registration and authentication
- Shopping cart
- Checkout flow
- Database integration

## Tech Stack

- HTML
- CSS
- JavaScript
- PHP
- MySQL

## How to run

1. Install a local server (e.g. XAMPP)
2. Copy the project folder to the server directory
3. Import the SQL file into the database
4. Start the server and access it through the browser
