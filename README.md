# Hextech E-commerce

E-commerce built as a technical high school final project in 2022. Frontend in HTML, CSS and JavaScript, backend in PHP and MySQL.

## Features

- Product catalog with individual product pages
- User registration and login
- Shopping cart
- Checkout and order tracking

## Tech Stack

- HTML, CSS, JavaScript
- PHP
- MySQL

## How to run

1. Install XAMPP
2. Copy the project folder to the server directory
3. Import the SQL file into the database
4. Start the server and open in the browser
